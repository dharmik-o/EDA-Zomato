Data Science Libraries: Core Functions

1. Pandas (Data Manipulation)

  • pd.read_csv() Reads a comma-separated values (CSV) file into a DataFrame.
  • pd.read_excel() Reads an Excel file into a pandas DataFrame.
  • pd.merge() Combines two DataFrames together based on a common column using database-style joins.
  • pd.concat() Concatenates multiple pandas objects along a particular axis.
  • pd.get_dummies() Converts categorical variables into dummy/indicator variables (One-Hot Encoding).
  • df.head() Returns the first n rows of the DataFrame.
  • df.columns Returns the column labels of the DataFrame.
  • df.info() Prints a concise summary of a DataFrame, including data types and non-null values.
  • df.describe() Generates descriptive statistics summarizing the central tendency, dispersion, and shape of a dataset.
  • df.isnull() / df.isna() Detects missing or null values in the DataFrame.
  • df.groupby() Groups DataFrame using a mapper or by a Series of columns to perform aggregated operations.
  • df.reset_index() Resets the index of the DataFrame to the default integer index.
  • df.rename() Alters axes labels or column headings.
  • df.drop() Removes specified labels from rows or columns.
  • df.unique() Returns unique values of a Series object.
  • df.value_counts() Returns a Series containing counts of unique values.
  • df.fillna() Fills NA/NaN values using a specified method or filler value.
  • df.apply() Applies a function along an axis of the DataFrame.
  • df.astype() Casts a pandas object to a specified standard data type.
  • series.map() Maps values of Series according to an input mapping or function.
  • series.str.split() Splits string values around a given separator/delimiter.
  • series.str.replace() Replaces occurrences of pattern/regex in the Series with some other string.

2. Matplotlib (Core Visualization)

  • plt.plot() Draws continuous lines, typically used for time-series data or mathematical functions.
  • plt.scatter() Creates a classic scatter plot mapping individual (x, y) coordinates.
  • plt.bar() / plt.barh() Renders standard vertical or horizontal bar charts.
  • plt.hist() Plots a classic histogram to show data frequency bins.
  • plt.pie() Generates a pie chart representing relative proportions.
  • plt.boxplot() Renders a box-and-whisker plot to highlight medians, quartiles, and outliers.
  • plt.imshow() Displays data as an image on a 2D regular raster.
  • plt.pcolormesh() Creates a pseudocolor plot with a non-regular rectangular grid.
  • plt.subplots() Creates a figure and a set of subplots for grid layouts.
  • plt.show() Displays all open figures.

3. Seaborn (Statistical Visualization)

Relational Plots
  • sns.scatterplot() Enhanced scatter plot supporting hue mapping, styling, and sizing based on variables.
  • sns.lineplot() Draws line graphs with automatic confidence intervals for aggregated data.
  • sns.relplot() Figure-level function to map relational plots onto a grid of subplots using facets.
Distribution Plots
  • sns.histplot() An enhanced version of a histogram with options for smoothing and stacking.
  • sns.kdeplot() Computes and plots a smooth, continuous kernel density estimate probability distribution.
  • sns.ecdfplot() Plots empirical cumulative distribution functions.
  • sns.rugplot() Draws marginal ticks along the axis to show individual data point locations.
Categorical Plots
  • sns.barplot() Shows point estimates and confidence intervals with rectangular bars.
  • sns.countplot() Shows the counts of observations in each categorical bin.
  • sns.boxplot() Shows distributions with respect to categories using box-and-whisker structure.
  • sns.violinplot() Combines box plots with density traces to show detailed distribution shapes.
  • sns.stripplot() / sns.swarmplot() Scatter plots adjusted specifically for categorical axes so points don't overlap blindly.
  • sns.catplot() The master figure-level function for categorical plots in grids.
Matrix & Multi-Plot Grids
  • sns.heatmap() Plots rectangular data as a color-encoded matrix.
  • sns.clustermap() A matrix plot that runs hierarchical clustering to group similar data visually.
  • sns.pairplot() Generates a grid showing pairwise relationships across numeric columns in a dataset.
  • sns.jointplot() Creates a bi-variate center plot flanked by uni-variate marginal distribution plots.
  • sns.FacetGrid() A framework allowing you to map any plotting function sequentially onto grids.
Regression Plots
  • sns.lmplot() / sns.regplot() Plots a scatter plot alongside a calculated linear regression line fit and confidence interval.

4. Scikit-Learn (Machine Learning Preprocessing)

  • LabelEncoder() Encodes target or categorical feature labels with value choices between 0 and n_classes-1.
  • OneHotEncoder() Encodes categorical features as a one-hot numeric array.
  • StandardScaler() Standardizes features by removing the mean and scaling to unit variance.
  • train_test_split() Splits arrays or matrices into random train and test subsets for validation.
  • .fit_transform() Fits data parameters (to scale/encode) and returns the transformed array simultaneously.
  • .transform() Applies calibrated scaling/encoding values to independent data (like a validation set).
